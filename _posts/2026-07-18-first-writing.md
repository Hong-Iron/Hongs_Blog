This is a sample post in the **writings** category, so you can see how the
editorial article layout looks with real prose: drop cap on the first
paragraph, comfortable reading width, and a serif body face.

Replace this file, or leave it as a template. New posts are easiest to
create with the CLI:

```
python3 scripts/blog.py post new "My Post Title" --category writings --tags life,notes
```

> Blockquotes get a left rule and italic serif treatment — useful for pull
> quotes in longer pieces.

Everything else is standard Markdown: headings, images, code fences, and
links all work as expected.
